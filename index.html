Learning Bower!

  • Install bower - npm install -g bower
  • Uninstall bower - npm uninstall -g bower
  • Update bower - npm update -g bower
  • bower install loadash -- Updates or installs - also loads dependent library
  • bower install loadash#3.4.0 -- Updates or installs to specific version
  • bower info loadash -- Displays info about lodash - all releases etc
  • bower list -- Displays all libraries in the project
  • bower unintall lodash -- Removes the lodash
  • bower update -- Updates entire project libaries to latest available - should not use - might break somethig
  • bower search lodash- Search library with lodash name in it - but to search for keyword need to go to bower.io
  • bower init- Creates bower.json file
  • bower install lodash --save- updated bower.json file with newly install library
  • bower home- Opens home URL in the browser
